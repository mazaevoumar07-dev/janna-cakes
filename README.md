## 🚀 Как начать

### 1. Установка (быстро!)
Просто откройте `index.html` в браузере.

### 2. Замена контактной информации

Откройте `index.html` и найдите раздел "Контакты":
```html
<p>+7 (XXX) XXX-XX-XX</p>
<a href="https://wa.me/7XXXXXXXXXX">
```

Замените:
- `+7 (XXX) XXX-XX-XX` - на ваш номер
- `7XXXXXXXXXX` - на ваш номер без спецсимволов
- `@janna_cakes` - на ваш Telegram

### 3. Добавление фотографий торов

1. Создайте папку `images` в папке проекта
2. Положите туда ваши фотографии (1.jpg, 2.jpg и т.д.)
3. Отредактируйте `script.js`:

Найдите эту строку:
```javascript
modalImage.src = `images/${item.id}.jpg`;
```

Раскомментируйте её (удалите `//` в начале).

Или просто замените весь блок в `script.js`:
```javascript
const galleryItems = [
	{ id: 1, name: 'Ваше название торта', image: 'images/1.jpg' },
	{ id: 2, name: 'Ваше название торта', image: 'images/2.jpg' },
	// ... и так далее для всех ваших фото
];
```

### 4. Изменение цветов

Откройте `styles.css` и найдите переменные в начале:
```css
:root {
	--primary-color: #d4a5a5;      /* Ваш основной цвет */
	--secondary-color: #f5f1f1;    /* Фоновый цвет */
	--accent: #c9959d;             /* Акцентный цвет */
}
```

Примеры цветов:
- Розовый: `#d4a5a5`, `#e6b8b8`
- Посейголд: `#e6c4a6`, `#d9b8a0`
- Кремовый: `#f5e6d3`, `#f0dcc4`
- Серо-голубой: `#a8bcc1`, `#b5c8ce`

### 5. Запуск локально

Установите простой сервер (опционально):

**Python 3:**
```bash
python -m http.server 8000
```

**Node.js:**
```bash
npx http-server
```

Затем откройте http://localhost:8000

## 🆘 Часто задаваемые вопросы

**Q: Как добавить свое лого?**
A: Замените "janna.cakes" в `navbar-brand` на `<img src="logo.png" alt="Logo">`

**Q: Как поменять название?**
A: Найдите "janna.cakes" в `index.html` и замените везде на ваше название

**Q: Сайт некрасиво выглядит по мобильному?**
A: Очистите кэш браузера (Ctrl+Shift+Del) и перезагрузите страницу
